\chapter{相关理论基础}\label{Chap:chap2}

本章围绕后续算法设计所需的关键概念给出相关理论基础。首先介绍马尔可夫决策过程（MDP）与Soft Actor-Critic（SAC）等基础框架，随后给出基于模型的离线强化学习建模方式、集成模型的不确定性估计方法；最后说明离线到在线训练范式及在线微调（online fine-tuning）的基本流程与常见稳定性问题，为第三章与第四章的算法描述提供统一的符号与概念基础。


\section{马尔可夫决策过程}\label{Sec2.1}

强化学习问题通常建模为马尔可夫决策过程（Markov Decision Process, MDP）\cite{Sutton2018RLBook}。MDP由六元组$M = (\mathcal{S}, \mathcal{A}, T, r, \mu_0, \gamma)$定义，其中$\mathcal{S}$表示状态空间，包含智能体可能处于的所有状态；$\mathcal{A}$表示动作空间，包含智能体可执行的所有动作；$r: \mathcal{S} \times \mathcal{A} \rightarrow \mathbb{R}$为奖励函数，量化状态-动作对的即时收益；$T(s'|s,a)$为状态转移概率函数，定义在状态$s$执行动作$a$后转移至状态$s'$的条件概率；$\mu_0$为初始状态分布；$\gamma \in (0,1)$为折扣因子，用于调节即时奖励与未来奖励的相对权重。

在MDP框架中，智能体的行为由策略$\pi(a|s)$刻画，表示在状态$s$下选择动作$a$的条件概率分布。强化学习的核心任务是寻找最优策略，为此需要定义价值函数以量化策略的长期累积收益。

状态价值函数$V_M^\pi(s)$定义为在MDP $M$中，从状态$s$出发并遵循策略$\pi$所能获得的期望折扣回报：
\begin{equation}\label{Eq:Chap2_value_function}
  V_M^\pi(s) = \mathbb{E}_{\pi, M}\left[\sum_{t=0}^{\infty} \gamma^t r(s_t, a_t) \mid s_0 = s\right],
\end{equation}
其中$s_t$和$a_t$分别表示时刻$t$的状态和动作，期望在策略$\pi$和环境动力学$M$下计算。状态价值函数衡量的是智能体处于特定状态时的长期预期收益。

动作价值函数$Q_M^\pi(s,a)$定义为在状态$s$执行动作$a$后，继续遵循策略$\pi$所能获得的期望折扣回报：
\begin{equation}\label{Eq:Chap2_q_function}
  Q_M^\pi(s,a) = \mathbb{E}_{\pi, M}\left[\sum_{t=0}^{\infty} \gamma^t r(s_t, a_t) \mid s_0 = s, a_0 = a\right],
\end{equation}
动作价值函数衡量的是在特定状态下执行特定动作的长期预期收益。

状态价值函数与动作价值函数之间存在如下关系：
\begin{equation}\label{Eq:Chap2_v_q_relation}
  V_M^\pi(s) = \mathbb{E}_{a \sim \pi(\cdot|s)}[Q_M^\pi(s,a)] = \sum_{a \in \mathcal{A}} \pi(a|s) Q_M^\pi(s,a),
\end{equation}
即状态价值等于该状态下所有动作价值在策略分布下的期望。反之，动作价值函数可通过贝尔曼期望方程表示为：
\begin{equation}\label{Eq:Chap2_bellman}
  Q_M^\pi(s,a) = r(s,a) + \gamma \mathbb{E}_{s' \sim T(\cdot|s,a)}[V_M^\pi(s')],
\end{equation}
即当前奖励与后继状态价值折扣期望之和。

为分析策略在状态-动作空间上的访问行为，引入折扣状态-动作访问分布$\rho_M^\pi(s,a)$，定义为策略$\pi$在MDP $M$中访问状态-动作对$(s,a)$的非归一化折扣频率：
\begin{equation}\label{Eq:Chap2_visit_distribution}
  \rho_M^\pi(s,a) = \pi(a|s) \sum_{t=0}^{\infty} \gamma^t P(s_t = s | \pi, M),
\end{equation}
其中$P(s_t=s | \pi, M)$表示从初始分布$\mu_0$出发，遵循策略$\pi$于时刻$t$到达状态$s$的概率。该分布刻画了策略的状态-动作空间覆盖特性：$\rho_M^\pi(s,a)$值较大的区域对应策略频繁访问的区域，反之则对应策略较少探索的区域。

基于上述定义，策略的期望折扣回报$\eta_M^\pi$作为评估策略整体性能的核心指标，可通过以下三种等价形式表示：
\begin{equation}\label{Eq:Chap2_expected_return}
  \eta_M^\pi = \mathbb{E}_{s_0 \sim \mu_0}[V_M^\pi(s_0)] = \mathbb{E}_{s_0 \sim \mu_0, a_0 \sim \pi(\cdot|s_0)}[Q_M^\pi(s_0, a_0)] = \mathbb{E}_{(s,a) \sim \rho_M^\pi} \left[ r(s, a) \right].
\end{equation}

第一种形式从初始状态价值的角度表述期望回报；第二种形式从初始状态-动作价值的角度表述，便于与基于Q函数的算法衔接；第三种形式从访问分布的角度表述，将期望回报理解为策略访问的所有状态-动作对上奖励的加权平均。MDP的优化目标是求解最大化期望折扣回报的最优策略$\pi^* = \arg\max_\pi \eta_M^\pi$。

归一化状态-动作访问分布定义为$\tilde{\rho}_M^\pi(s,a) = (1 - \gamma) \cdot \rho_M^\pi(s,a)$，满足$\sum_{s,a}\tilde{\rho}_M^\pi(s,a) = 1$，构成状态-动作空间上的概率分布。在离线强化学习中，训练数据的分布$\rho_D$与目标策略的访问分布$\rho_M^\pi$之间的差异构成分布偏移问题的根源：当目标策略访问的状态-动作对在训练数据中覆盖不足时，价值函数的估计将产生显著偏差。


\section{Soft Actor-Critic算法}\label{Sec2.2}

Soft Actor-Critic（SAC）\cite{Haarnoja2018SAC}是一种基于最大熵框架的Actor-Critic算法，在连续控制任务中表现优异，是本文后续章节策略优化的基础算法。SAC的核心思想是在最大化期望回报的同时最大化策略的熵，从而鼓励探索并提升策略的鲁棒性。

SAC的优化目标定义为带熵正则的期望回报：
\begin{equation}\label{Eq:Chap2_sac_objective}
  J(\pi) = \sum_{t=0}^{\infty} \mathbb{E}_{(s_t, a_t) \sim \rho_\pi}\left[r(s_t, a_t) + \alpha_{ent} \mathcal{H}(\pi(\cdot|s_t))\right],
\end{equation}
其中$\mathcal{H}(\pi(\cdot|s)) = -\mathbb{E}_{a \sim \pi}[\log \pi(a|s)]$为策略在状态$s$下的熵，$\alpha_{ent} > 0$为温度参数，控制熵正则项的相对重要性。熵正则项的引入使策略倾向于在高价值动作之间保持一定的随机性，避免过早收敛到确定性策略，有助于在训练过程中保持探索能力。

SAC采用Actor-Critic架构，包含三个核心组件：策略网络（Actor）$\pi_\varphi$、双Q值网络（Critic）$Q_{\theta_1}, Q_{\theta_2}$和目标Q值网络$Q_{\bar{\theta}_1}, Q_{\bar{\theta}_2}$。

Q值网络通过最小化软贝尔曼残差进行训练：
\begin{equation}\label{Eq:Chap2_sac_critic}
  \mathcal{L}(\theta_i) = \mathbb{E}_{(s, a, r, s') \sim \mathcal{D}}\left[\left(Q_{\theta_i}(s, a) - y\right)^2\right], \quad i = 1, 2,
\end{equation}
其中目标值$y$定义为：
\begin{equation}\label{Eq:Chap2_sac_target}
  y = r + \gamma \left(\min_{j=1,2} Q_{\bar{\theta}_j}(s', a') - \alpha_{ent} \log \pi_\varphi(a'|s')\right), \quad a' \sim \pi_\varphi(\cdot|s').
\end{equation}

采用双Q网络的最小值可以有效抑制Q值的过估计问题\cite{Hasselt2016DoubleDQN}。目标网络通过软更新$\bar{\theta} \leftarrow \tau \theta + (1 - \tau) \bar{\theta}$与主网络保持同步，其中$\tau \ll 1$为软更新系数，确保训练过程的稳定性。

策略网络通过最大化以下目标进行优化：
\begin{equation}\label{Eq:Chap2_sac_actor}
  J(\varphi) = \mathbb{E}_{s \sim \mathcal{D}, a \sim \pi_\varphi}\left[\min_{j=1,2} Q_{\theta_j}(s, a) - \alpha_{ent} \log \pi_\varphi(a|s)\right],
\end{equation}
该目标同时考虑动作价值和策略熵：第一项驱动策略选择高价值动作，第二项鼓励策略保持多样性。对于连续动作空间，策略网络通常输出高斯分布的均值和方差，通过重参数化技巧\cite{Kingma2014VAE}实现可微采样：
\begin{equation}\label{Eq:Chap2_reparameterization}
  a = \tanh(\mu_\varphi(s) + \sigma_\varphi(s) \odot \epsilon), \quad \epsilon \sim \mathcal{N}(0, I),
\end{equation}
其中$\tanh$函数将动作限制在有界区间内，$\odot$表示逐元素乘法。温度参数$\alpha_{ent}$可通过自动调节机制学习\cite{Haarnoja2018SAC}，使熵保持在目标水平附近：
\begin{equation}\label{Eq:Chap2_alpha_loss}
  \mathcal{L}(\alpha_{ent}) = \mathbb{E}_{a \sim \pi_\varphi}\left[-\alpha_{ent} \left(\log \pi_\varphi(a|s) + \bar{\mathcal{H}}\right)\right],
\end{equation}
其中$\bar{\mathcal{H}}$为目标熵，通常设为$-\dim(\mathcal{A})$。当策略熵低于目标熵时，$\alpha_{ent}$增大以鼓励探索；当策略熵高于目标熵时，$\alpha_{ent}$减小以促进收敛。

SAC在离线强化学习中的应用需要特别注意分布偏移问题：由于无法与环境交互获取反馈，Q值网络对分布外动作的估计可能产生严重的过估计。本文后续章节通过不确定性惩罚机制约束策略在模型预测可靠的区域内优化，有效缓解这一问题。


\section{基于模型的离线强化学习框架与不确定性度量}\label{Sec2.3}

在离线强化学习场景中，智能体仅能访问由行为策略$\pi_D$预先采集的静态数据集$\mathcal{D}_{off} = \{(s_i, a_i, s_i', r_i)\}_{i=1}^N$，无法与真实环境进行在线交互。基于模型的离线强化学习方法通过监督学习从离线数据中构建环境转移模型$\hat{T}_{\theta}(s', r|s, a)$，该模型以当前状态$s$和动作$a$为输入，预测下一状态$s'$及即时奖励$r$的联合分布。相比无模型方法直接学习价值函数或策略，基于模型的方法具有更高的样本效率：一旦学习到准确的环境模型，策略可通过与模型的虚拟交互进行优化，而无需访问真实环境。

转移模型通常采用参数化高斯分布进行建模：
\begin{equation}\label{Eq:Chap2_transition_model}
  \hat{T}_{\theta}(s', r|s, a) = \mathcal{N}(\mu_{\theta}(s, a), \Sigma_{\theta}(s, a)),
\end{equation}
其中$\mu_{\theta}(s, a)$为条件均值函数，$\Sigma_{\theta}(s, a)$为条件协方差矩阵，均由参数为$\theta$的神经网络拟合。采用高斯分布建模的优势在于：均值参数可刻画状态转移的确定性成分，协方差参数可刻画转移的随机性成分，同时输出概率分布便于后续的不确定性量化。在实际应用中，为简化计算和保证数值稳定性，通常采用对角协方差矩阵，即假设各维度的预测误差相互独立。

模型参数通过最大化离线数据的对数似然进行优化：
\begin{equation}\label{Eq:Chap2_model_loss}
  \mathcal{L}(\theta) = -\mathbb{E}_{(s,a,s',r) \sim \mathcal{D}_{off}}[\log \hat{T}_{\theta}(s', r|s, a)].
\end{equation}

展开高斯分布的对数似然，训练损失等价于加权的均方误差与对数方差之和，其中均方误差项驱动模型学习状态转移的确定性规律，对数方差项约束模型输出合理的不确定性估计。这种概率建模方式使得模型不仅能预测转移的期望值，还能量化预测的置信度。

完成模型学习后，策略可通过与学习得到的转移模型交互生成虚拟轨迹进行优化，无需访问真实环境。具体而言，从离线数据集中采样初始状态，利用当前策略与转移模型进行短期推演（rollout），生成虚拟轨迹数据。这些虚拟数据可用于训练价值函数或直接优化策略，从而实现无需环境交互的策略改进。这一特性使基于模型的方法具有更高的样本效率，尤其适用于数据采集成本高昂的实际应用场景。

然而，学习得到的转移模型与真实环境之间不可避免地存在模型误差。模型误差可分解为两个来源：偏差（Bias）和方差（Variance）。偏差源于模型类（如神经网络）的表达能力限制，可能无法精确拟合真实环境的复杂动力学；方差源于有限训练数据导致的参数估计不确定性，在数据稀疏的区域尤为显著。当策略访问离线数据覆盖不足的状态-动作区域时，模型预测的可靠性下降，外推误差将随模型推演步数累积放大。设单步模型误差为$\epsilon_m$，推演$H$步后的累积误差约为$O(H \cdot \epsilon_m)$，在长时间范围任务中可能导致严重的策略性能退化。因此，如何量化并控制模型不确定性、抑制分布外区域的过估计，是基于模型离线强化学习的核心挑战。

集成方法（Ensemble Methods）通过训练多个独立初始化的模型，利用模型间预测的分歧度量化不确定性\cite{Lakshminarayanan2017Ensemble}。设集成包含$N$个模型$\{f_{\phi}^1, \ldots, f_{\phi}^N\}$，每个模型在相同数据集上独立训练但采用不同的随机初始化，则预测不确定性可通过集成预测的统计特性进行估计。

强化学习中的不确定性可从来源角度分为两类\cite{Charpentier2022Uncertainty}。认知不确定性（Epistemic Uncertainty）源于模型对环境知识的缺乏，反映训练数据覆盖不足导致的模型不确定性，可通过收集更多数据逐步降低。偶然不确定性（Aleatoric Uncertainty）源于环境固有的随机性，反映状态转移过程本身的内在噪声，无法通过增加数据量消除。

对于离线到在线强化学习，认知不确定性具有特殊重要性：高认知不确定性区域对应离线数据覆盖不足的状态-动作空间，恰恰是在线探索应优先关注的目标区域。通过识别并引导策略探索这些高不确定性区域，可有效补充转移模型的训练数据，提升模型在全状态-动作空间的泛化能力。

在基于模型的强化学习中，集成动力学模型的每个成员$\hat{T}_{\phi}^i$输出高斯分布$\mathcal{N}(\mu_{\phi}^i(s,a), \Sigma_{\phi}^i(s,a))$。其中，单个模型的预测方差$\Sigma_\phi^i(s,a)$反映偶然不确定性，刻画环境动力学的内在随机性。为捕捉认知不确定性，可利用集成模型均值预测的分歧度定义如下度量：
\begin{equation}\label{Eq:Chap2_disvar}
  \text{DisVar}(s,a) = \frac{1}{d}\sum_{j=1}^d \text{Var}_{i=1,\ldots,N}[\mu_{\phi,j}^i(s,a)],
\end{equation}
其中$d$为预测向量的维度，$\mu_{\phi,j}^i(s,a)$表示第$i$个模型预测均值的第$j$个分量，$\text{Var}_{i=1,\ldots,N}[\cdot]$表示在$N$个集成成员间计算方差。该度量的核心思想是：在数据充分覆盖的区域，各模型收敛至相近的预测，分歧度较低；在数据稀疏的区域，模型因初始化差异而产生显著不同的预测，分歧度较高。本文将在算法设计中融合两类不确定性信息，实现更高效的策略优化与在线探索。



\section{离线到在线训练范式与在线微调}\label{Sec2.4}

离线到在线强化学习（Offline-to-Online RL）通常采用“离线预训练—在线微调”的两阶段范式：首先在离线数据集上完成策略与价值函数（或动力学模型）的预训练，得到具备基本可行性且相对稳定的初始化策略；随后在受限的在线交互预算下采集少量新数据并继续更新策略，实现对真实环境的快速适配\cite{Lee2022O2O,Song2023HybridRL}。该范式兼具离线学习的低风险与在线学习的可适配性，尤其适用于交互成本高昂、存在安全约束或上线试错代价较大的场景。

从优化目标角度看，离线到在线设定可理解为在交互预算约束下求解最优策略。设离线数据集为$\mathcal{D}_{off}=\{(s,a,r,s')\}$，由行为策略$\pi_D$预先采集得到；在线阶段可额外获得由当前策略与环境交互产生的回放池$\mathcal{D}_{on}$。在给定交互预算$B$的条件下，离线到在线学习的目标可形式化为：
\begin{equation}\label{Eq:Chap2_o2o_budget}
  \max_{\pi}\ \eta_M^{\pi}\ \ \ \text{s.t.}\ \ |\mathcal{D}_{on}| \le B,
\end{equation}
其中$\eta_M^{\pi}$为策略的期望折扣回报，$|\mathcal{D}_{on}|$表示在线交互获得的数据量。该形式强调了两阶段范式的本质矛盾：离线阶段需要在不访问真实环境的条件下获得足够稳定的初始化能力，而在线阶段则需要在有限样本下实现可靠提升。

为兼顾“保持离线知识”与“吸收在线反馈”，离线到在线方法通常采用混合重放机制：训练批次同时从$\mathcal{D}_{off}$与$\mathcal{D}_{on}$采样\cite{Lee2022O2O,Zhao2023E2O}。若记在线采样比例为$\beta_t \in [0,1]$，则可将训练样本的来源写为
\begin{equation}\label{Eq:Chap2_mixed_replay}
  (s,a,r,s') \sim \beta_t\,\mathcal{D}_{on} + (1-\beta_t)\,\mathcal{D}_{off},
\end{equation}
其中$\beta_t$可随训练步数$t$动态调度。直观而言，较小的$\beta_t$有助于在微调初期维持训练分布的连续性，降低价值函数估计在分布突变下的震荡；而随着在线数据覆盖逐步扩展，增大$\beta_t$能够提高新数据对策略更新的贡献，避免长期停留在离线数据的局部最优。

离线到在线微调的首要风险是稳定性退化，典型表现为预训练策略在在线阶段初期出现性能骤降，即灾难性遗忘\cite{Lee2022O2O}。其根源在于：离线预训练得到的价值函数或模型在分布外区域缺乏可靠监督，若在线阶段立即放松约束并大幅改变策略分布，Critic对新访问区域的估计误差会被Actor更新放大并进一步改变访问分布，形成正反馈回路。围绕该问题，现有方法主要从价值估计保守化与策略更新受控两条路径缓解。其一，O2O\cite{Lee2022O2O}与E2O\cite{Zhao2023E2O}等方法通过Q集成、取最小值或逐步减少集成规模等机制，在微调早期维持偏悲观的价值估计，以抑制过估计导致的策略崩塌；Cal-QL\cite{Nakamoto2024CalQL}则从价值校准角度改善离线价值偏差在在线阶段的放大问题。其二，AWAC\cite{Nair2020AWAC}通过优势加权回归使策略更新更接近离线初始化分布，PEX\cite{Zhang2023PEX}采用离线策略与在线策略的混合形式以平滑过渡，从而在“保守—灵活”之间实现可控的权衡。

除稳定性外，交互效率同样是离线到在线范式的关键矛盾：在线预算有限，若在线采样长期停留在与离线数据高度重合的区域，新增样本的边际效用较低，难以带来显著改进；但若采样策略过于激进，又会过早进入模型或价值估计不可靠的区域并诱发性能波动。因此，在线阶段需要更具目标性的采样机制，以在可控风险下优先采集“覆盖不足但与高回报相关”的状态-动作区域。近年来的趋势是引入不确定性、信息增益等信号作为采样线索：在离线阶段，不确定性更常被视为需要规避的风险；而在在线阶段，不确定性也可以转化为指示“最值得补充数据”的探索信号。特别地，对于基于模型的方法，集成动力学模型提供的认知不确定性能够刻画模型覆盖不足区域；若进一步结合价值信息，则可将探索从“盲目增加覆盖”转向“面向高回报区域的定向补齐”，为后续定向探索与高效微调提供背景基础。


\section{本章小结}\label{Sec2.5}

本章给出了本文所需的相关理论基础。首先在MDP框架下定义了策略、价值函数与访问分布，并介绍了作为后续策略优化基座的SAC算法；随后描述了基于模型的离线强化学习框架与集成模型不确定性估计方法，为不确定性惩罚与探索信号的构造提供形式化工具；最后围绕离线到在线训练范式，给出离线预训练、在线交互与混合采样的基本流程，讨论了微调阶段的稳定性风险与交互预算下的数据效率问题，并强调不确定性信息在两阶段中可分别承担“保守约束”与“探索线索”的作用。上述内容为第三章的纯离线算法设计与第四章的离线到在线微调算法设计奠定了概念与符号基础。
